# bili鹅 - 使用说明

## 项目简介

bili鹅是一款专为电视端设计的B站第三方客户端应用，采用Kotlin语言开发，提供流畅的大屏视频观看体验。

## 功能特性

### 核心功能
- ✅ 用户登录（扫码登录/游客模式）
- ✅ 视频浏览与搜索
- ✅ 高清视频播放（支持720P/1080P）
- ✅ 弹幕显示
- ✅ 遥控器适配
- ✅ 个人中心管理
- ✅ 观看历史记录

### 界面特色
- 📺 专为电视大屏优化
- 🎮 遥控器友好操作
- 🎨 简洁美观的UI设计
- ⚡ 流畅的交互体验

## 安装说明

### 方法一：直接安装APK
1. 下载 `app-release.apk` 文件
2. 将APK文件拷贝到U盘或通过网络传输到电视设备
3. 在电视上使用文件管理器找到APK文件
4. 点击安装（可能需要在设置中开启"未知来源"安装权限）

### 方法二：通过Android Studio安装
1. 打开Android Studio
2. 导入本项目
3. 连接电视设备（通过ADB或USB调试）
4. 点击运行按钮安装到设备

### 方法三：命令行安装
```bash
# 连接设备
adb connect <电视IP地址>

# 安装APK
adb install app-release.apk
```

## 使用指南

### 遥控器操作

#### 基本导航
- **方向键（上/下/左/右）**：在界面元素间移动焦点
- **确定键（OK）**：选择当前焦点项
- **返回键**：返回上一级页面
- **主页键**：返回应用主页

#### 视频播放控制
- **确定键**：播放/暂停
- **左/右键**：快退/快进（10秒）
- **上/下键**：调节音量
- **菜单键**：显示播放设置菜单

### 功能详解

#### 1. 登录功能
- **扫码登录**：使用手机B站APP扫描二维码登录
- **游客模式**：不登录直接浏览（部分功能受限）

#### 2. 首页推荐
- 显示B站推荐视频
- 支持上拉加载更多
- 点击视频封面进入播放

#### 3. 视频搜索
- 支持关键词搜索
- 提供热门搜索词
- 搜索历史记录

#### 4. 视频播放
- 自动选择最佳画质
- 支持弹幕开关
- 播放进度记忆
- 手势控制（音量、亮度、进度）

#### 5. 个人中心
- 查看用户信息
- 观看历史
- 收藏夹管理
- 设置选项

#### 6. 设置功能
- 默认画质设置
- 弹幕设置
- 缓存管理
- 关于应用

## 开发指南

### 环境要求
- Android Studio Arctic Fox (2020.3.1) 或更高版本
- Kotlin 1.8+
- Android SDK API 21+ (Android 5.0+)
- JDK 8 或更高版本

### 项目结构
```
bili鹅/
├── app/
│   ├── src/main/java/com/biligo/
│   │   ├── ui/                    # UI界面层
│   │   ├── data/                  # 数据层
│   │   ├── domain/                # 业务逻辑层
│   │   └── utils/                 # 工具类
│   ├── build.gradle.kts          # 应用构建配置
│   └── AndroidManifest.xml       # 应用清单

└── settings.gradle.kts           # 项目设置
```

## 常见问题

### Q1: 无法安装APK？
**A:** 请检查：
1. 电视是否开启了"未知来源"安装权限
2. APK文件是否完整下载
3. 设备存储空间是否充足

### Q2: 视频无法播放？
**A:** 可能原因：
1. 网络连接不稳定
2. 视频已被删除或版权限制
3. 尝试切换画质或重新登录

### Q3: 遥控器无响应？
**A:** 请检查：
1. 遥控器电池是否有电
2. 遥控器是否正确配对
3. 尝试重启应用或设备

### Q4: 弹幕不显示？
**A:** 请检查：
1. 弹幕开关是否开启
2. 网络连接是否正常
3. 视频是否支持弹幕

### Q5: 登录失败？
**A:** 请检查：
1. 网络连接是否正常
2. 二维码是否过期（5分钟内有效）
3. 手机B站APP是否为最新版本
