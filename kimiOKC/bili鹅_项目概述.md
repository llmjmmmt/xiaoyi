# bili鹅 - B站第三方电视版应用

## 项目简介
bili鹅是一款专为电视端设计的B站第三方客户端应用，采用Kotlin语言开发，提供流畅的大屏视频观看体验。

## 核心功能

### 1. 用户认证模块
- **登录功能**: 支持B站账号登录（扫码登录/账号密码登录）
- **注册功能**: 新用户注册B站账号
- **用户信息管理**: 显示用户头像、昵称、等级等信息
- **登录状态保持**: 自动保存登录凭证

### 2. 视频内容展示
- **首页推荐**: 展示B站推荐视频内容
- **分区浏览**: 动画、游戏、生活、音乐等分区
- **搜索功能**: 支持关键词搜索视频
- **视频详情**: 显示视频标题、UP主、播放量、弹幕等信息

### 3. 视频播放功能
- **高清播放**: 支持720P/1080P高清播放
- **弹幕显示**: 实时显示弹幕内容
- **播放控制**: 播放/暂停、快进/快退、调节音量
- **全屏模式**: 沉浸式全屏观看体验
- **播放历史**: 记录观看进度

### 4. 用户体验优化
- **遥控器适配**: 专为电视遥控器优化
- **语音控制**: 支持语音搜索（可选）
- **收藏功能**: 收藏喜欢的视频
- **观看历史**: 查看观看记录

## 技术架构

### 开发语言
- **Kotlin**: 主要开发语言
- **Jetpack Compose**: 现代UI框架

### 核心依赖库
- **ExoPlayer**: 视频播放器
- **Retrofit**: 网络请求
- **Coil**: 图片加载
- **Room**: 本地数据库
- **DataStore**: 数据存储

## 开发环境要求
- Android Studio Arctic Fox 或更高版本
- Kotlin 1.8+
- Android SDK API 21+ (Android 5.0+)
- JDK 8 或更高版本

## 目标设备
- Android TV 5.0+
- 电视盒子
- 智能电视
- 投影仪（Android系统）

## 应用特色
1. **大屏优化**: 专为电视大屏设计的用户界面
2. **遥控器友好**: 完全适配遥控器操作
3. **流畅播放**: 优化的视频播放体验
4. **简洁界面**: 清晰直观的导航结构
5. **快速启动**: 应用启动速度快
